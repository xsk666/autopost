## 项目构成

简单介绍一下这个项目的构成
> 全部代码在[`main`](/main)文件夹中  
> 部署文档在[`course`](/course)

> 启动文件是[`index.py`](/main/index.py)  
> 用户数据是[`users.json`](/main/users.json)  
> 登陆模块是[`sign.py`](/main/sign.py)   
> 获取打卡数据是[`getinfo.py`](/main/getinfo.py)  
> 打卡程序是[`post.py`](/main/post.py)
